# 들어가며

나는 11년차 맥북 유저지만 한 번도 수리를 맡긴 적이 없다. 애플 케어가 너무 비쌌고 첫 맥북은 애플 케어 플러스까지 먹여서 애지중지 사용했지만 고장이 없었다. 두 번째 맥북부터는 그냥 고장나면 다시 사면 된다 마인드로 살았다.

이번 맥북은 중고로 구매했다. 16인치 M1 프로, 램 32gb, 디스크 1Tb 로 가장 파워풀한 성능이다. 신품을 살 예산은 없지만 고성능은 쓰고 싶은데, 애플 케어 플러스도 달려있어서 큰 고민 없이 구매했다.
거래를 마치고 집에 와서 세팅을 하다보니 아뿔싸! 몇 가지 사소한 문제가 있었다.

1. J 키가 너무 쉽게 눌린다.
2. 스피커 음질이 이상하다.

증상이 사소해서 그냥 뭍어두고 사용하고 있었다. 애플 케어 만료 이틀 전까지.
만료되기 전에 수리를 맡길 겸 가까운 잠실 애플 스토어로 갔다. 말도 많고 탈도 많은 애플 AS 서비스. 과연 괜찮을까?

# 방문 전에 타임머신 백업

수리를 맡기기 전에 중요한 데이터를 백업하라고 한다. 나의 경우 추가 외장 디스크가 없었다. 나의 경우 Windows Samba 서버를 활용했다. Windows Samba 서버 만드는 방법은 쉽게 찾을 수 있다. 그 후에 마운트하고 타임머신 할 때 도움이 됐던 글을 공유한다.
[SMB를 사용하는 macOS Time Machine 백업 서버 만들기](https://kycfeel.github.io/2017/06/20/smb%EB%A5%BC-%EC%82%AC%EC%9A%A9%ED%95%98%EB%8A%94-%EB%A7%A5-Time-Machine-%EB%B0%B1%EC%97%85-%EC%84%9C%EB%B2%84-%EB%A7%8C%EB%93%A4%EA%B8%B0/)

1. Windows에서 Samba 서버를 만든다.
2. Mac 에서 마운트한다.
3. Samba 공간에 Mac의 디스크 파일을 만든다.

```bash
# 네트워크 드라이브로 타임머신 백업 허용
sudo defaults write com.apple.systempreferences TMShowUnsupportedNetworkVolumes 1
# Sparsebundle 동적 디스크 생성
sudo hdiutil create -size 1024g -type SPARSEBUNDLE -fs "HFS+J" TimeMachine.sparsebundle
# 타임머신 백업 경로 강제 지정
sudo tmutil setdestination /Volumes/TimeMachine
```

5. 방금 만든 디스크 파일을 Mac에 마운트한다.
6. 타임머신으로 백업을 수행한다.

```bash
# 타임머신 백업 시작 전에 성능 스로틀 해제
sudo sysctl debug.lowpri_throttle_enabled=0
```

# 수리 맡기기

수리를 맡기기 전에 [애플 지니어스 바 예약 사이트](https://getsupport.apple.com/)에서 방문 예약을 해야한다.
예약을 하지 않으면 상담까지 시간이 걸릴 수 있다. 잠실점은 오후 10시까지 운영했지만 여유롭게 오후 시간에 방문했다.

스피커의 이상 증상을 설명하기 애매했다. 일단 소리를 들어보면 100% 싸구려 음색이 나오지만 노래에 따라 다르기도 하고 설명하기가 난감하다. 나의 경우 알림 사운드중에 Boop 을 기준으로 생각하고 방문했다.
점원분이 친절하게 상담해주셨지만 처음엔 소리는 이상하지 않다고 생각하셨다. 그 후 부팅 과정에서 진입하는 하드웨어 점검 페이지에서 종합 점검을 수행했다.
스피커 점검 차례에서 삐-- 하는 소리가 나는데 이걸 들어보시더니 교체를 해야 될 것 같다고 했다.

맥북을 맡기고 수리비 8만 9천원을 결제했다.
수리 대상은 맞지만 부품이 없어서 부품 배송까지 시간이 걸린다고 부품이 오면 다시 방문해달라고 했다.

# 추가로 발견된 배터리 결함

사흘 뒤 스토어로부터 전화가 왔다. `뭐야이번호` 라는 앱에서 잠실 애플스토어를 스팸이라고 표기해서 하마터면 못받을뻔했다.

점검해보니 배터리에 주름이 생겨서 정상 사용이 불가능하다고.
배터리와 상판을 모두 교체해야 될 것 같고 수리를 진행해도 되냐는 연락이었다. **금액은 무료로!**
J키가 쉽게 눌리는 결함이 배터리에 주름이 원인이었던 것 같다.

# 결과

노트북을 수령하고 상태를 확인하니 스피커도 모두 정상이고 배터리도 100%가 찍히고 있었다. J키 눌림도 정상이었다.
우려했던게 무안할 정도로 수리 경험은 매우 만족스러웠다. 백업이 조금 귀찮았고 그 외로는 딱히 불편한 과정이 없었다.
결과가 좋으면 다 좋으니까. 새로운 배터리로 다시 태어났으니 아끼면서 사용해야겠다.
